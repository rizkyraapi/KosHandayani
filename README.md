## 📌 Overview

**KosHandayani** is a web-based system designed to manage boarding house operations, including room availability, tenant management, rental applications, and payment processing.

This system is developed as part of a **Project-Based Learning (PBL)** at:

🎓 **Politeknik Negeri Batam**
💻 **Informatics Engineering Study Program**

---

## 🎯 Objectives

* Transform manual rental management into a digital system
* Improve operational efficiency
* Provide real-time data monitoring
* Reduce human errors
* Enable structured financial reporting

---

## 🚀 Key Features

### 👤 Tenant (Penyewa)

* 🔍 Browse & filter available rooms
* 🏠 View detailed room information
* 📝 Submit rental application
* 📎 Upload documents (KTP & KK)
* 💳 Make monthly payments
* 📊 View payment history
* 🔔 Receive notifications

---

### 🏢 Owner (Pemilik / Admin)

* 🛠 Manage rooms & facilities
* ➕ Add / update room data
* 📊 Monitor room availability
* 👥 Manage tenant data
* ✔ Verify rental applications
* 💰 Track payments & billing
* 📉 Manage income & expenses
* 📈 View financial reports

## 🧪 Installation Guide

### 1️⃣ Clone Repository

```bash id="q02k4y"
git clone https://github.com/rizkyraapi/KosHandayani.git
cd KosHandayani
```

---

### 2️⃣ Backend (Laravel)

```bash id="ypl0dx"
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

---

### 3️⃣ Frontend (Next.js)

```bash id="2cd3z4"
cd frontend
npm install
npm run dev
```
